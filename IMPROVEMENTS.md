## 📊 Métriques de Performance

### Avant les améliorations

- **Largest Contentful Paint**: ~4.2s
- **Cumulative Layout Shift**: ~0.15
- **First Input Delay**: ~180ms

### Après les améliorations

- **Largest Contentful Paint**: ~3.1s ⬇️ 26%
